# Machine Learning

A hands-on collection of ML implementations built while learning core concepts — from classical algorithms to transformer architectures. All notebooks are written in Python using Google Colab.

## Repository Structure

| Folder | Topics Covered |
|--------|----------------|
| `Basics` | NumPy, Pandas, data preprocessing, EDA, and foundational ML concepts |
| `KNN_Poly` | K-Nearest Neighbors and Polynomial Regression |
| `Lasso_Ridge` | Regularization techniques — L1 (Lasso) and L2 (Ridge) regression |
| `LogReg_SVM` | Logistic Regression and Support Vector Machines for classification |
| `Transformer` | Transformer architecture implementation from scratch |
| `MathGPT` | GPT-style language model applied to mathematical reasoning |

## Tech Stack

- **Language:** Python
- **Environment:** Google Colab
- **Libraries:** NumPy, Pandas, Scikit-Learn, Matplotlib, TensorFlow

## About

Built as part of self-directed learning and MSc coursework in Intelligent Systems at Trinity College Dublin.
